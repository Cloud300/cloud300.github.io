使用Github搭建免费图床

步骤

  1. 首先新建一个Github仓库，仓库名自定义

  2. 生成token（计算机术语，令牌）

  3. Github右上角头像处→设置→列表里的：Developer settings→personal access tokens → 选择：Generate new token（生成新令牌）

  4. 名称随便。下面是仓库权限，都勾上。最后是下面的绿色小框框，生成令牌



  1. 生成之后就可以看到 token 页面，它只会显示一次，复制令牌保存到文档里，一会配置要用



配置PicGo

  1. 先安装软件:下载,有条件的可以挂代理下载。

  2. 安装完后开始配置，我的配置如下：



  • 仓库名：刚才新建的仓库名称。格式为：用户名/仓库名。注意大小写跟空格，仓库地址输入不正确会出现服务端报错
  • 设定分支名：master
  • Token：刚才的令牌，复制粘贴即可
  • 指定存储路径：我的设置为 /001，意思是在仓库根目录下新建一个名为001的文件夹，并把上传的图片放到此文件夹下。
  • 设定自定义域名：https://cdn.jsdelivr.net/gh/ 用户名/仓库名。。注意大小写跟空格。
  • 配置完成后点击确定
  • 左侧的上传区，把图片拖进去，重命名就可以了。
  • 完结撒花🙃🙃
